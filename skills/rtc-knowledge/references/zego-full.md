## 四、SDK API 体系

### 4.1 客户端核心 API

```
ZegoExpressEngine.createEngine()     → 创建引擎
engine.loginRoom()                   → 登录房间
engine.logoutRoom()                  → 登出房间
engine.startPublishingStream()       → 推流
engine.stopPublishingStream()        → 停止推流
engine.startPlayingStream()          → 拉流
engine.stopPlayingStream()           → 停止拉流
engine.startPreview()                → 预览本地画面
engine.stopPreview()                 → 停止预览
engine.setVideoSource()              → 设置视频源（摄像头/屏幕共享）
engine.enableCamera()                → 开关摄像头
engine.muteMicrophone()              → 静音麦克风
engine.setEventHandler()             → 设置回调
```

## 五、计费体系

### 5.1 音视频时长后付费单价

| 计费类型 | 分辨率档位 | 单价（元/千分钟） |
|----------|-----------|------------------|
| 高音质纯音频 | — | **7.00** |
| 标清视频 SD | ≤360P (640×360) | **12.00** |
| 高清视频 HD | ≤720P (1280×720) | **25.00** |
| 超清视频 HD+ | ≤1080P (1920×1080) | **98.00** |
| 超清视频 2K | ≤1440P (2560×1440) | **112.00** |
| 超清视频 4K | ≤2160P (3840×2160) | **252.00** |

### 5.2 计费规则

- **按拉流计费**：按用户实际拉取音视频流的时长统计
- **视频含音频**：同时拉视频和音频时，仅计视频费用
- **多流分别计算**：拉多条视频流时，分别统计，不叠加分辨率
- **不拉流不计费**：在房间但未拉流，不收费
- **精度**：秒级统计，分钟级出账（不足1分钟按1分钟算）

### 5.3 套餐包

- 支持预付费通用套餐包
- 具体面额和折扣需联系销售或查看活动页
- 2025年活动：全线产品1折起

### 5.4 免费额度

- 官方文档未明确标注每月免费额度
- 新用户通常有试用额度（需联系销售确认）

## 六、技术架构特点

1. **QQ基因**：创始团队来自腾讯QQ音视频，技术底蕴深厚
2. **腾讯投资**：C轮腾讯领投，与腾讯有资本关系
3. **自研z264编码器**：相同条件下画质优于标准x264
4. **AI视频引擎**：自研AI引擎支持超分、暗光增强、背景分割、插帧
5. **平台覆盖最广**：业内首批支持HarmonyOS(ArkTS)，支持Unreal/Unity游戏引擎
6. **直播方案灵活**：超低延迟/CDN Plus/CDN三档方案平滑切换
7. **万人上麦**：支持万人同时上麦音视频通话
8. **99%+秒开率**：直播场景首帧优化
9. **抗80%丢包**：弱网优化能力强
