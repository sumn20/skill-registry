## 二、产品线

### 2.1 核心产品

| 产品 | 说明 |
|------|------|
| **实时音视频 (RTC)** | 语音/视频通话 + 互动直播，SDK 4.x 最新版 |
| **极速直播** | 低延迟直播（比标准CDN直播延迟更低） |
| **实时消息 (RTM)** | 消息通道、呼叫、聊天、状态同步 |
| **云端录制** | 服务端录制音视频通话/直播内容 |
| **互动白板** | 多人实时协作白板 |
| **水晶球 (Agora Analytics)** | 全链路质量监控平台 |
| **对话式AI引擎** | 2025年3月发布，全球首个对话式AI引擎 |
| **实时码流加速** | 自定义码流在互联网上实时传输 |

### 2.2 对话式 AI 引擎（2025年重点）

- 支持任意文本大模型升级为"能说会道"的多模态对话大模型
- 兼容 DeepSeek、ChatGPT 等全球主流大模型
- 语音对话延迟中位数 **650ms**
- 定价：5000分钟 ¥469元起

## 三、产品功能矩阵

| 功能 | 说明 |
|------|------|
| **1v1单聊** | 点对点音视频通话 |
| **多人群聊** | 多人同时加入频道互动 |
| **观众连麦** | 观众与主播连麦 |
| **跨频道连麦 (PK)** | 多主播跨频道互动PK |

| **屏幕共享** | 共享屏幕/窗口/指定区域 |
| **美颜** | 基础美白、磨皮、祛痘、红润 |
| **自定义美颜/滤镜** | 配合自定义视频源，接入第三方美颜库 |
| **高清视频** | 最高1080p, 60fps |
| **自定义视频源/渲染器** | 支持非摄像头的自定义视频源 |

### 3.2 音频功能

| 功能 | 说明 |
|------|------|
| **纯语音通话** | 仅音频流 |
| **伴奏混音** | 本地/在线音频与人声混音 |
| **音频特效** | 变声、混响等 |
| **3A处理** | AEC + ANS + AGC |
| **原始音频数据** | 获取音频原始数据进行自定义处理 |

### 3.3 媒体服务

| 功能 | 说明 |
|------|------|
| **云端录制** | 云端录制频道音视频内容 |
| **本地服务端录制** | 本地服务器录制 |
| **旁路推流 (CDN)** | 推送到RTMP服务器/CDN |
| **输入在线媒体流** | 将在线媒体流导入频道 |
| **转码服务** | 云端转码 + RTMP网关 |
| **跨频道媒体流转发** | 将一个频道的流转发到其他频道 |

### 3.4 特殊说明

- **Web SDK 不支持修改原始音视频数据**（Native SDK 支持）
- 标清(SD)档位声网未单独列出，HD起步 ≤720P

## 四、SDK API 体系

### 4.1 客户端 SDK (RTC 4.x)

```
AgoraRTC.createClient()          → 创建客户端
client.join()                    → 加入频道
client.leave()                   → 离开频道
AgoraRTC.createMicrophoneAudioTrack() → 创建麦克风音频轨道
AgoraRTC.createCameraVideoTrack()     → 创建摄像头视频轨道
client.publish([localTrack])     → 发布本地流
client.subscribe(remoteUser)     → 订阅远端流
client.on("user-published")      → 监听远端用户发布事件
client.on("user-unpublished")    → 监听远端用户取消发布
AgoraRTC.createScreenVideoTrack()     → 屏幕共享
```

## 六、技术架构特点

1. **SD-RTN™**：声网自建全球实时通信网络（Software Defined Real-time Network）
2. **海外优势**：全球部署近100个数据中心，近千台服务器
3. **按频道人数计时**：而非"按流计时"，同等场景下费用更可控
4. **AI战略转型**：2025年重点押注对话式AI引擎，作为第二增长曲线
5. **生态开放**：与多家大模型厂商兼容（DeepSeek/ChatGPT等）
6. **编解码**：支持H.264/H.265/VP8/VP9/AV1
