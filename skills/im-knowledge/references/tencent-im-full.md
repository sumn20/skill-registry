## 一、产品概述

腾讯云即时通信 IM（Instant Messaging）基于 QQ 和微信 20+ 年技术积累，提供全球覆盖的即时通讯 PaaS 服务。支持单聊、群聊、聊天室、社群等多场景，覆盖社交、电商、在线教育、企业办公、游戏、直播等行业。

### 核心优势
- **QQ/微信同源技术**：经过数十亿用户验证的底层架构
- **全球接入**：中国+亚洲+欧洲+北美+南美+大洋洲+非洲全球节点
- **全平台覆盖**：Android/iOS/HarmonyOS/Web/Windows/Mac/Flutter/RN/Unity/UE
- **IM+TRTC 一体化**：IM 套餐内直接购买音视频通话能力
- **微信小程序深度互通**：原生支持小程序 IM SDK
- **社群能力**：支持 10 万人社群，企业版可达 100 万人

## 四、完整功能列表

### 4.1 消息类型
- 文本消息、图片消息（≤28MB）、表情消息
- 语音消息（≤28MB，含时长）、短视频消息（≤100MB）
- 文件消息（≤100MB，不限格式）
- 地理位置消息、自定义消息（红包/石头剪刀布等）
- 系统通知消息、群 Tips 消息
- 合并消息（≤300条）

### 4.2 消息功能
- 消息撤回（默认2分钟内）、消息转发、消息删除
- 已读回执（旗舰版+）、消息扩展（旗舰版+）
- @功能、正在输入状态
- 本地搜索（好友/群组/消息）
- 云端搜索（全局/指定会话/用户/逻辑搜索）
- 全员推送 / 标签推送 / 属性推送
- 离线推送（APNs/小米/华为/OPPO/vivo/Google FCM/魅族）
- 多端同步、漫游消息

### 4.3 群组类型

| 类型 | 成员上限 | 特性 |
|------|----------|------|
| 好友工作群（Work） | 专业200/旗舰2000/企业6000 | 仅好友邀请，无需审批 |
| 陌生人社交群（Public） | 同上 | 需群主/管理员审批 |
| 临时会议群（Meeting） | 同上 | 随意进出，查看入群前消息 |
| 直播群（AVChatRoom） | **无上限** | 不支持历史消息 |
| 社群（Community） | 10万（企业版100万） | 话题分组，大规模社区 |

### 4.4 关系链与用户资料
- 好友管理：查找/申请/添加/删除/校验
- 好友分组：创建/删除/重命名
- 黑名单：添加/移除/校验
- 用户资料：昵称/头像/性别/签名/位置+20个自定义字段
- 好友自定义资料（最多20个字段）

### 4.5 账号与登录
- 批量导入/删除/查询账号
- 多端登录策略：单平台/双平台/三平台/全平台
- 用户在线/离线状态管理

### 4.6 安全与审核
- 云端审核：文本/图片/语音/视频智能识别
- 敏感词过滤
- 账号禁用

### 4.7 运营工具
- IM 控制台：应用管理/统计分析/回调配置
- 数据统计：DAU/注册数/消息量/在线数等
- 实时监控仪表盘
- 插件市场（云端搜索/投票/接龙/翻译/语音转文字等）
- 私有化部署支持

## 七、与 TRTC 联动优势

1. **IM 套餐内直购音视频**：无需单独开通 TRTC，IM 控制台一键购买
2. **信令互通**：IM 信令 SDK 与 TRTC 房间管理无缝对接
3. **TUICallKit 组件**：几行代码集成 1v1/群组音视频通话
4. **一个 SDK 双能力**：IM SDK + TRTC SDK 统一管理
5. **账号体系统一**：同一套 UserID + UserSig，无需二次鉴权
6. **录制+审核统一**：IM 消息审核 + TRTC 录制在同一控制台管理
7. **计费统一**：一张账单，一个控制台
